# AI Resume Intelligence Platform

An AI-powered Resume Intelligence Platform that analyzes resumes, extracts important information, evaluates ATS compatibility, and provides intelligent career insights using NLP and AI techniques.

---

## 🚀 Features

- Resume Upload System
- AI-Based Resume Analysis
- ATS Score Evaluation
- Skill Extraction
- Career Guidance Pages
- Roadmap Suggestions
- Resume Processing & Parsing
- User-Friendly Interface

---

## ⚙️ Installation

### Clone Repository

```bash

cd ai-resume-intelligence-platform
```

---

## 📦 Install Dependencies

```bash
pip install -r requirements.txt
```
